# redaktura-skills

Набор скиллов для редактуры, статей и UX-текстов на русском языке.

Метод основан на практиках Люды Сарычевой ([gladlax.ru](https://gladlax.ru)) и собран совместно с Никитой Архиповым ([niar42.com](https://niar42.com)).

## Скиллы

| Скилл | Что делает |
|---|---|
| `redpolitika` | Создаёт и обновляет `.agents/redpolitika.md`: читатель, площадки, регистр, словарь, фактура |
| `redaktura` | Редактирует готовые тексты: смысл, структура, тональность, предложения, слова, ритм |
| `statya` | Ведёт статью с нуля: бриф, фактура, структура, черновик, самопроверка |
| `ux-copy` | Пишет и правит тексты интерфейса: кнопки, ошибки, формы, пустые состояния, уведомления |

## Как устроена работа

Сначала настраивается `redpolitika`. Она создаёт файл `.agents/redpolitika.md`, который читают остальные скиллы.

```
redpolitika -> redaktura
             -> statya
             -> ux-copy
```

Если редполитики нет, скиллы всё равно работают. Для постоянной работы с одним проектом лучше её создать.

## Установка

Скопируйте нужные папки скиллов в директорию Codex:

- `$CODEX_HOME/skills/`
- `~/.codex/skills/`, если `CODEX_HOME` не задан

Можно копировать весь набор целиком или только нужные папки: `redpolitika`, `redaktura`, `statya`, `ux-copy`. Учтите: `statya` и `ux-copy` на финальной проверке ссылаются на справочники `redaktura`, поэтому её лучше ставить всегда.
